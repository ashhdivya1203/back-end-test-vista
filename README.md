# Backend - Vista Technical Test

## 🚀 Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/ashhdivya1203/back-end-test-vista.git
   cd back-end-test-vista

2. Install dependencies

    npm install

3. Start the server

    npm start

⚠️ The server will run on http://localhost:4000


📂 API Endpoints

Companies

-GET /companies → Get all companies

-POST /companies → Create a new company (name, registration number)

Services

-GET /services → Get all services

-POST /services → Create a new service (name, description, price, company)

🛠 Tech Stack

-Node.js

-Express.js

-MongoDB / Mongoose (if DB is used)
